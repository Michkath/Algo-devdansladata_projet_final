Architecture du pipeline
01
Python · Requests
Scraping
Collecte brute des données hébergements depuis data.gouv.fr via scrapping.py.
02
Python · Pandas
Nettoyage
Normalisation, déduplication et validation des données dans cleaning.py. Stockage dans MongoDB.
03
PostgreSQL · Star Schema
Data Warehouse
Chargement dans un DW structuré via dw_loader.py.
04
Flask · REST
API REST
Exposition des données via endpoints filtrables et paginés. Dockerisé.

Retourne la liste des hébergements avec filtres et pagination.

ParamètreTypeDéfautDescription
villestring—Filtrer par commune
typestring—Type d'hébergement

per_pageint20Résultats par page (max 100)
